## 1. 概述

本主板是一款专为**机械臂运动控制**设计的嵌入式控制板。核心采用独立的 **STM32F103C8T6** 单片机，负责运动学解算、舵机 PWM 驱动、传感器数据采集及通信管理。板载丰富的通信接口（USB、CAN、UART、SWD、蓝牙）和扩展接口（OLED、电阻触摸屏、手柄、舵机），支持与 **ROS（机器人操作系统）** 主控协同工作。

### 关键特性

| 特性 | 说明 |
|------|------|
| 主控芯片 | STM32F103C8T6 (ARM Cortex-M3, 72MHz, 64KB Flash, 20KB SRAM) |
| USB 转串口 | CH9102 (USB 2.0 to UART, QFN 封装) |
| 下载方式 | Micro-USB 一键串口下载 + SWD 调试接口 |
| 舵机接口 | 多路 PWM 排针输出，可驱动多个舵机 |
| 通信总线 | USB, CAN, UART, I2C/SPI, 蓝牙 |
| 显示外设 | OLED (I2C/SPI) + 电阻触摸屏 |
| 电源 | 外部 DC 输入 → 3.3V/5V 稳压，5V@5A 对外供电，12V 预留输出 |

### 2.1 核心控制单元: STM32F103C8T6

| 属性 | 值 |
|------|-----|
| 型号 | **STM32F103C8T6** |
| 封装 | **LQFP48** (48 引脚) |
| 内核 | ARM Cortex-M3, 72 MHz |
| Flash | 64 KB |
| SRAM | 20 KB |
| GPIO | 37 个可用 I/O |
| 定时器 | 4×16 位通用定时器 + 2×高级定时器 (可生成多路 PWM) |
| 通信接口 | 2×SPI, 2×I2C, 3×USART, 1×USB 2.0 Full-Speed, 1×CAN 2.0B |
| ADC | 2×12 位 ADC (10 通道) |

**角色**：全板核心。负责：
- 机械臂正/逆运动学解算
- 多路舵机 PWM 信号生成
- 串口通信（与 ROS 主控交互）
- 传感器读取（ADC/I2C）
- 显示驱动（OLED/触摸屏）
- 用户交互（按键、蜂鸣器）

### 2.2 USB 转串口芯片: CH9102

| 属性 | 值 |
|------|-----|
| 型号 | **CH9102** |
| 封装 | QFN |
| 功能 | USB 2.0 全速 ↔ TTL UART 桥接 |
| 用途 | 固件一键串口下载 (ISP) + 与 ROS 主控（树莓派/Jetson 等）通信 |

**工作原理**：
- USB 端连接 Micro-USB 接口 (H2)
- UART 端连接 STM32 的 USART1 (PA9/PA10)
- 配合"程序下载控制跳线帽"，可通过 DTR/RTS 信号自动控制 STM32 的 BOOT0 和 RESET，实现**一键 ISP 下载**（无需手动切换启动模式）
- 正常运行时作为串口透传通道，实现 ROS 主控 ↔ STM32 的指令和数据交换

### 2.3 电源管理系统

| 元件 | 说明 |
|------|------|
| **电源输入接口** | 黑色 DC 桶型插座 (5.5mm×2.1mm)，接受外部直流电源 |
| **电源开关** | 红色船型开关，物理切断主供电 |
| **5V/5A 对外供电** | 白色 USB-A 母座，可对外输出 5V@5A，常用于给树莓派、摄像头等供电 |
| **电压转换** | 板上电感 (L1/L2/L3)、电解电容 (C26/C27 等)、贴片电容组成开关电源/线性稳压电路，将输入电压降压为 3.3V (供 STM32) 和 5V (供舵机/外设) |
| **LDO 推测** | 常见选择为 AMS1117-3.3 或类似低压差线性稳压器，具体需看原理图 |

### 2.4 下载与调试接口

| 接口 | 类型 | 功能 |
|------|------|------|
| **Micro-USB (H2)** | Micro-USB 母座 | 通过 CH9102 实现串口一键下载 + ROS 主控通信 |
| **SWD (H1)** | 4 针排针 (GND, SWCLK, SWDIO, 3V3) | 标准 ARM SWD 接口，用于 ST-Link 调试和高速烧录 |
| **程序下载控制跳线帽** | 跳线帽 (2 针) | 控制 BOOT0 电平，切换正常运行模式 / Bootloader 模式 |

### 2.5 通信与外设接口

| 接口 | 类型 | 协议 | 功能 |
|------|------|------|------|
| **CAN 接口** | 白色连接器 | CAN 2.0B | 工业级抗干扰总线，连接分布式驱动器或传感器节点 |
| **蓝牙接口** | 排针 | UART | 连接 HC-05/HC-06 蓝牙模块，实现无线遥控/数传 |
| **OLED 显示屏接口** | 排针 | I2C / SPI | 0.96"/1.3" OLED 显示屏，显示系统状态 |
| **电阻屏接口** | 白色连接器 | ADC + GPIO | 4 线/5 线电阻触摸屏，提供图形化人机交互 |
| **舵机接口** | 长排针 | PWM | 多路舵机驱动接口 (VCC, GND, Signal)，用于机械臂关节 |
| **蜂鸣器 (BUZ1)** | 圆形元件 | GPIO/PWM | 声音提示（按键音、错误报警） |
| **蜂鸣器控制跳线帽** | 跳线帽 | - | 连接/断开蜂鸣器 |

### 2.6 人机交互

| 元件 | 说明 |
|------|------|
| **复位按键** (Reset) | 蓝色按钮，触发 STM32 硬件复位 (NRST) |
| **用户按键** (User) | 蓝色按钮，可编程功能（启动/停止、模式切换等） |

### 3.1 背面接口

| 接口 | 类型 | 功能 |
|------|------|------|

| **预留 12V 电源输出端口** | 白色 JST/XH2.54 插座 | 输出 12V DC，可驱动外部 12V 电机、继电器、电磁阀等 |

| 元件 | 封装 | 推测功能 |
|------|------|---------|

```
    │                         │   │               │

    │                                         │

    │  舵机 1..N ◄── PWM (TIM) ────────────────┘
    │
    └──── 对外: 5V@5A USB-A / 12V 预留输出
```

### 典型工作流程

1. **上电**：外部 DC 输入 → 电源开关 → 3.3V LDO / 5V Buck 稳压
2. **固件加载**：Micro-USB 连接 PC → CH9102 串口 → STM32 Bootloader → 烧录固件
3. **ROS 通信**：树莓派/Jetson USB → CH9102 → STM32 USART → 接收运动指令
4. **运动控制**：STM32 解析指令 → 运动学解算 → 多路 PWM 输出 → 舵机转动
5. **人机交互**：OLED 显示状态 / 触摸屏操作 / 手柄遥控 / 按键切换模式 / 蜂鸣器提示

## 6. 开发与调试注意事项

### 6.1 固件烧录

- **串口一键下载**：跳线帽短接"程序下载控制"，通过 CH9102 + DTR/RTS 自动进 Bootloader，使用 `stm32flash` 或 STM32CubeProgrammer 烧录
- **SWD 调试**：使用 ST-Link V2 或 DAP-Link，连接 SWD 接口 (GND/SWCLK/SWDIO/3V3)

### 6.2 与 ROS 集成

典型的 ROS 通信架构：
```
ROS 主控 (树莓派/Jetson)
  │ USB
  ▼
CH9102 → STM32 USART
  │
  ▼
自定义串口协议 (位置/速度/力矩指令)
  │
  ▼
STM32 运动学解算 → PWM 驱动舵机
```

### 6.3 电源注意事项

- 总电流取决于舵机数量和负载，务必匹配外部电源
- 5V5A 对外供电和舵机共用电源轨时需考虑总功率
- 12V 预留输出独立于主 5V/3.3V 轨
