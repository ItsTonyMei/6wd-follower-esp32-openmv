# 接线文档 — 6轮车-UniBoard

> 更新时间：2026-04-28
> 系统组成：ESP32（统一控制）+ ESP8266 板（仅供电分配）+ OpenMV（视觉）

---

## 1. 系统架构

```
┌─────────────────────────────────────────────────────────┐
│                      7.4V 电池                           │
└─────────────────────────┬───────────────────────────────┘
                          │
           ┌──────────────┴──────────────┐
           ▼                             ▼
  ┌───────────────┐              ┌───────────────┐
  │   RZ7886      │              │  XL2596 降压板 │
  │  电机驱动板   │              │  (5V 输出)    │
  │  (7.4V直通)   │              └───────┬───────┘

          │                    ┌─────────┴─────────┐
          │                    ▼                    ▼
          │            ┌──────────────┐     ┌───────────┐

          │                   │            → OpenMV UART1 RX
          │                   │                    │
          ▼                   ▼                    ▼

```

## 2. ESP8266 板的新角色：供电分配板

ESP8266 NodeMCU **不再运行任何代码**，仅利用其排针做 5V/GND 分配：

```
XL2596 5V ──→ ESP8266 VIN ──→ ESP8266 板上 5V 总线
                              ├──→ 左 HC-SR04 VCC
                              ├──→ 右 HC-SR04 VCC
                              └──→ RZ7886 逻辑 5V

ESP8266 GND ──→ ESP8266 板上 GND 总线
              ├──→ 左 HC-SR04 GND
              ├──→ 右 HC-SR04 GND
              ├──→ RZ7886 GND
              └──→ ESP32 GND（共地）
```

- ESP8266 的 D1-D8 GPIO 排针空置，不再连接任何信号线
- ESP8266 不需要烧录代码，上电后 5V 总线直通
- ESP8266 自身耗电 ~70mA，在 XL2596 能力范围内

## 3. ESP32 → RZ7886 电机驱动

| ESP32 GPIO | RZ7886 端子 | 功能 |
|------------|-------------|------|
| GPIO25 | A-1 | 左电机前进 |
| GPIO26 | A-2 | 左电机后退 |
| GPIO27 | B-1 | 右电机前进 |
| GPIO19 | B-2 | 右电机后退 |
| GND | GND | 共地 |

> 所有 GPIO 均使用 ESP32 LEDC PWM，无启动约束。

---

## 4. ESP32 → HC-SR04 超声波传感器 ×2

| ESP32 GPIO | HC-SR04 左 | HC-SR04 右 |
|------------|------------|------------|
| GPIO32 | TRIG | - |
| GPIO34 (仅输入) | ECHO（经分压） | - |
| GPIO33 | - | TRIG |
| GPIO35 (仅输入) | - | ECHO（经分压） |

### ECHO 电平分压（必须）

HC-SR04 在 5V 供电时 ECHO 输出 ~5V，ESP32 GPIO 不保证 5V tolerant：

```
HC-SR04 ECHO ──┤1.8kΩ├──┬── ESP32 GPIO (34/35)
                         │
                       ┤3.3kΩ├
                         │
                        GND
```

分压后 ≈ 3.25V。GPIO34/35 是仅输入引脚，无内部上拉，正好适配经分压后的 ECHO 信号。

---

## 5. ESP32 ↔ OpenMV UART

| OpenMV | ESP32 | 说明 |
|--------|-------|------|
| P4 (TX) | GPIO15 (RX) | OpenMV 单向发送 VIS 数据 |
| P5 (RX) | 不接 | 单向通信 |
| GND | GND | 共地 |

OpenMV 通过 USB 供电，3.3V TTL 电平，无需电平转换。

---

## 6. 接线步骤

### 步骤 1：共地

所有设备 GND 连在一起：ESP32 GND → ESP8266 GND → RZ7886 GND → HC-SR04 GND → OpenMV GND

### 步骤 2：ESP8266 供电分配

- XL2596 5V → ESP8266 VIN
- ESP8266 5V → 左 HC-SR04 VCC
- ESP8266 5V → 右 HC-SR04 VCC
- ESP8266 5V → RZ7886 逻辑 5V（如已从其他地方取电可不接）

### 步骤 3：ESP32 → RZ7886

| ESP32 | → | RZ7886 |
|-------|---|--------|
| GPIO25 | → | A-1 |
| GPIO26 | → | A-2 |
| GPIO27 | → | B-1 |
| GPIO19 | → | B-2 |
| GND | → | GND |

### 步骤 4：ESP32 → HC-SR04

- 左：GPIO32 → TRIG，ECHO 经分压 → GPIO34
- 右：GPIO33 → TRIG，ECHO 经分压 → GPIO35

### 步骤 5：ESP32 ↔ OpenMV

- OpenMV P4(TX) → ESP32 GPIO15(RX)
- GND 共地
- OpenMV USB 供电

## 7. 与旧方案的接线差异

| 变化 | 旧方案 | 新方案 |
|------|--------|--------|
| 电机信号 | ESP8266 D1-D4 | ESP32 GPIO25/26/27/19 |
| 超声波信号 | ESP8266 D5-D8 | ESP32 GPIO32/34/33/35 |
| ESP32↔ESP8266 UART | GPIO16/17 连接 | 不连接（ESP8266 仅供电） |
| ESP8266 GPIO | 控制电机和传感器 | 空置 |
| HC-SR04 供电 | ESP8266 5V 排针 | 同前（ESP8266 仍分配 5V） |
